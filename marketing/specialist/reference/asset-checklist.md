# Reference: Pre-flight checklist (run before Joshua sees it)

## Every asset
- [ ] Facts pulled from the canon: email, phone, URLs, tagline all match `positioning-canon/README.md`.
- [ ] No em-dashes or en-dashes anywhere, including signatures and captions.
- [ ] Recognition arc: opens with the reader's tension, not our features.
- [ ] One idea, one ask. "Vorschlag, kein Pitch."
- [ ] Voice matches the register (`marketing/voice/`), read aloud once.
- [ ] No unapproved claims; no sweeping privacy or compliance guarantees.

## Print only
- [ ] 3mm bleed baked in, correct pixel size at 300dpi.
- [ ] sRGB, density tagged 300, delivered as JPG.
- [ ] Full-bleed colors reach the edge; safe margin on text; square corners.
- [ ] QR present and test-scanned.

## Web only
- [ ] HTML and the i18n object both updated, both dash-free.
- [ ] Language toggle works; no console errors.
- [ ] Responsive checked; dark sections and contrast fine; alt text present.
- [ ] Canonical links only.

## Gate
- [ ] Rendered and visually verified.
- [ ] Put in front of Joshua. Nothing publishes or prints without his yes.
