# Assets — fertige Artefakte

Die konkreten Sachen, die rausgehen. **Jedes Asset zieht Stimme aus `../voice/` und Story aus
`../narratives/` — nicht neu erfinden.**

- `decks/` — PowerPoints / Präsentationen
- `html/` — interaktive HTML-Dokumente (Vorbild: der DAA-Brief — HTML schlägt PDF nach Gesprächen)
- `one-pagers/` — kompakte Übersichten (Print/PDF)
- `templates/` — wiederverwendbare Asset-Vorlagen (Deck-Gerüst, One-Pager-Gerüst, HTML-Shell)

> Tipp: Das DAA-DAA-Template (blau/gelb) liegt als Beispiel in
> `../../clients/daa/deliverables/`. Für generische Vorlagen hier eine entkundete Version ablegen.
